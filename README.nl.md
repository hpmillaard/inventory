# Inventory

Eenvoudige repository met een kleine webinterface en PowerShell-scripts voor inventorybeheer.

Korte omschrijving

- Webpagina om de inventory te bekijken en te openen via `index.html`.
- PowerShell-scripts voor export en weergave van inventorygegevens.

Belangrijke bestanden

- `index.html` — eenvoudige webinterface/voorkant.
- `install.ps1` — (optioneel) installatie- of setup-hulp.
- `Inventory.ps1` — hoofdscript voor het exporteren/aanmaken van inventory.
- `view-Inventory.ps1` — script om inventory lokaal te bekijken.

Gebruik

1. Open `index.html` in een browser voor de eenvoudige UI.
2. Of run de PowerShell-scripts in PowerShell:

```powershell
.\install.ps1
.\Inventory.ps1
.\view-Inventory.ps1
```
