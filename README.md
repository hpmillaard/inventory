# Inventory

Simple repository with a small web interface and PowerShell scripts for inventory management.

Short description

- Web page to view the inventory via `index.html`.
- PowerShell scripts to export and view inventory data.

Key files

- `index.html` — simple web interface.
- `install.ps1` — (optional) installation/setup helper.
- `Inventory.ps1` — main script to export/create inventory.
- `view-Inventory.ps1` — script to view inventory locally.

Usage

1. Open `index.html` in a browser to use the simple UI.
2. Or run the PowerShell scripts in PowerShell:

```powershell
.\install.ps1
.\Inventory.ps1
.\view-Inventory.ps1
```
